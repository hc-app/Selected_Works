# Serial and parallel fetcher weather api

/////////////////////////////////////////////////////////////////////

## Serial fetcher program:

Compile the file "serial.cpp":
g++ serial.cpp -o serial

Execute the serial fetcher program with 'input.txt' in the same directory:
./serial input.txt

/////////////////////////////////////////////

## Parallel fetcher program:

Compile the file "parallel.cpp":
g++ parallel.cpp -o parallel

Execute the parallel fetcher program with 'input.txt' in the same directory:
./parallel input.txt
