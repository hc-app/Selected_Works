# Storing Data Using Parallel Threads

//////////////////////////////////////////////////////////////////////////////////////////

The program generates random number as data in parallel and stores them in a binary search tree. The program checks if a random generated number is already in the BST. If the number is already in the BST it skips that number and displays a message instead of inserting the duplicate node. 

//////////////////////////////////////////////////////////////////////////////////////////

How to run the program:

Program: 
threads.cpp

Compile the file "threads.cpp":
`g++ threads.cpp -o threads`

Execute the program. `<number>` is the number of threads the program will create:
`./threads <number>`
