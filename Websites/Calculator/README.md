# Website: Calculator (HTML5/CSS/Javascript React)

The website contains a calculator application that displays the inputs and the running total. It can take input from keys or mouse clicks. The website is made from HTML5, CSS, and Javascript with React.

## The project showcases the following features:

### Javascript:
- React functional components, states and hooks
- Async/synchronous calls and promises
- Key mapping and parsing
- Event-listeners that triggers on key presses and execute application functionality
- Functional programming reusing code to keep the project code simple to understand and reduce project file size
- Programming control flow and logic of application's functionality use case and edge cases
- Regex
- CSS updated on event triggers
- Built using React to showcase extend of React knowledge and capability

### HTML5:
- Logically structured div and sections
- Organized and labeled HTML elements
- CDN imports on header to create the perception of smoothly loading

### CSS:
- Variable media screen size for desktop and mobile usability
- Button toggles
- Use of flex and grid displays
- Reusable CSS elements
