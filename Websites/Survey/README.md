# Website: Survey Form (HTML5/CSS)

The website contains a survey form. The website is made from HTML5 and CSS.

## The project showcases the following features:

### HTML5:
- Logically structured div and sections
- Organized and labeled HTML elements
- CDN imports on header to create the perception of smoothly loading

### CSS:
- Variable media screen size for desktop and mobile usability
- Slider and button toggles
- Use of flex and grid displays
- Reusable CSS elements
